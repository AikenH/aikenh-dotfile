## Keyboard Shortcuts

Leader Key: SPC

| Key1 | Key2 | Function|
| -- | -- | -- |
|  | V | 缩进 |
|  | S-V | 反向缩进 |
| <leader> | ev | 编辑vimrc|
| <leader> | ez | 编辑zshrc|
| <leader> | es | vimrc生效|
| <leader> | el | 删除文末空格|
| <leader> | er | tab换成spc|
| <leader> | w(hjkl) | 窗口切换 |
| <leader> | t(sqnp) | tabs管理、关闭、新窗口、切换|
| <leader> | sd | show diff |
| <leader> | ln | 相对行号切换 |
| <leader> | lr | 行号显示与否 |
| F5 | | 编译运行代码 |
| F6 | | 添加文件头 |
